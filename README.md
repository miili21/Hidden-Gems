# Hidden-Gems
Estrategia de Contenido de Netflix: Análisis de Popularidad, Recepción Crítica y Potencial de Subvaloración 
La investigacion se centra en comprender wue caracteristicas determinan el exito de peliculas y series en Netflix, ssi como detectar titulos subvalorados (hidden gems) que realmete pueden tener gran potencial de ser buenas producciones.

Este proyecto esta realizado en herramientas para el analisis de datos como R, generacion de un informe en RMarkdown, una visualizacion estrategica y estetica mediante un Dashboard (Streamlit) y la demostracion de todo lo realizado en este miso repositorio usando las buenas practicas de investigacion.
